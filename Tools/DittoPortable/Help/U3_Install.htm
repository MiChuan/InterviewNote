Thank you for installing Ditto for U3 Devices.

 

    What is Ditto?
 
Ditto is an extension to the Windows Clipboard.  You copy something to the Clipboard and Ditto takes what you copied and stores it in a database to retrieve at a later time.
    Basic Usage
   
  • Run Ditto.
  • Copy things to the clipboard, e.g. using Ctrl-C with text selected in a text editor.
  • Open Ditto by clicking its icon in the system tray or by pressing its Hot Key which defaults to Ctrl + ` – i.e. hold down Ctrl and press the back-quote (tilde ~) key.
  • Double click or press enter on the item to paste it to the previous window.
    U3 Device Specific


Changes from standard Ditto

  • Settings are now stored in a .ini file on the device 
  • Database is stored and ran from the device
  • Name copy and Paste are not supported 
  • New method for tracking focused window
  • Defaulted maximum clips to 75
  • Receiving clips over the network is off by default
  • CF_DIB (bitmaps) was removed from the default save clipboard types.  This was because of the large size if CF_DIB.

How to move your existing Ditto database to the U3 device

  1. Install ditto on you U3 device
  2. Run Ditto and set the maximum clips accordingly.  The default for U3 devices is 75, so if you have more than that in your existing database this setting will need to be increased. 
  3. Copy your database to the U3 device.  
  4.       Normally your existing database will be in the folder 'C:\Documents and Settings\<User Name>\Application Data\Ditto'
  5.       Copy that database to '<U3 drive>:\System\Apps\33820de7-c934-490a-8445-3212afbd9bf9\Data'
  6. The database must be named 'Ditto.db'
  7. Start Ditto on the U3 device.
