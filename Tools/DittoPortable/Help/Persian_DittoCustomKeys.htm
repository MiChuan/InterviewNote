Ditto - کلیدهای میانبر دلخواه

1) کلیدهای میانبر دلخواه

    

بعضی برنامه‌ها کلید میانبر ctrl-v را برای درج نمی‌پذیرند.  از نسخه‌ی 3.15.1 می‌توان برای هر برنامه کلید میانبر دلخواهی جهت درج در آن استفاده کرد.

 

2) چطور کلید میانبر دلخواه را تنظیم کنیم؟

  1. تنظیمات مربوط به درج

    1. از منوی شروع در برنامه Run بنویسید regedit و تایید کنید.

    2. به "HKCU\programs\Ditto\PasteStrings" بروید.

    3. مقداری از نوع string به نام برنامه‌ی مورد نظرتان اضافه کنید؛ باید نام کامل را وارد کنید؛ مثلا notepad.exe

    4. ترکیب کلیدهای مورد نظرتان را بنویسید 

  2. تنظیمات مربوط به نسخه‌برداری

    1. از منوی شروع در برنامه Run بنویسید regedit و تایید کنید.

    2. به "HKCU\programs\Ditto\CopyStrings" بروید.

    3. مقداری از نوع string به نام برنامه‌ی مورد نظرتان اضافه کنید؛ باید نام کامل را وارد کنید؛ مثلا notepad.exe

    4. ترکیب کلیدهای مورد نظرتان را بنویسید 

  3. تنظیمات مربوط به انتقال

    1. از منوی شروع در برنامه Run بنویسید regedit و تایید کنید.

    2. به "HKCU\programs\Ditto\CutStrings" بروید.

    3. مقداری از نوع string به نام برنامه‌ی مورد نظرتان اضافه کنید؛ باید نام کامل را وارد کنید؛ مثلا notepad.exe

    4. ترکیب کلیدهای مورد نظرتان را بنویسید 

3) نحوه‌ی نوشتن کلیدهای ترکیبی

مثال:‪  ^p  ‬(ctrl-p را می‌فرستد)

مثال:‪  %e{DELAY 50}p  ‬(Alt-e را می‌فرستد، 50 هزارم ثانیه صبر می‌کند و p را می‌فرستد)

مثال:‪  ^c  ‬(ctrl-c را می‌فرستد)

مثال:‪  ^x  ‬(ctrl-x را می‌فرستد)

قالب دستور عمل
{VKEY X} Sends the VKEY of value X.

Very useful if you don't want to recompile CSendKeys and add new Vkey to the hardcoded special keys table.

برای مثال {VKEY 13} معادل VK_RETURN است.
{BEEP X Y}} پخش بوق با فرکانس X به مدت Y هزارم ثانیه.
{DELAY X} کلید بعدی را با تاخیر X هزارم ثانیه می‌فرستد. بعد از این تاخیر کلیدهای بعدی بدون تاخیر فرستاده می‌شوند مگر اینکه مقدار پیش‌فرضی وجود داشته باشد (DELAY=X را ببینید).

مثال: {DELAY 1000} <-- ۱ ثانیه تاخیر برای کلید بعدی.
{DELAY=X} تاخیر پیش‌فرض را به اندازه X هزارم ثانیه تنظیم می‌کند. این تنظیم سبب می‌شود که هر کلید X هزارم ثانیه تاخیر داشته باشد.

اگر مقداری تنظیم شود و از {DELAY Y} استفاده کنید، کلیدی که در ادامه نوشته شود با Y هزارم ثانیه تاخیر فرستاده می‌شود اما کلیدهای بعدی تاخیر X هزارم ثانیه‌ای خواهند داشت.

 

مثال: {DELAY=1000} <-- همه‌ی کلیدهای بعدی تاخیر 1 ثانیه‌ای خواهند داشت.

 
{APPACTIVATE WindowTitle} برنامه را با استفاده از WindowTitle آن فعال می‌کند.

اگر بخواهید کلیدهای مختلفی را به برنامه‌های مختلفی بفرستید این گزینه بسیار سودمند است.
